# hendo3.github.io

Site estático em clima cyberpunk/terminal, publicado via GitHub Pages.

## Stack

- HTML
- CSS
- JavaScript

## O que tem

- Layout responsivo com setores em grade
- Logs aleatórios em tempo real
- Barra de comando estilo terminal
- Links ocultos no HTML e resolvidos em runtime

## Como rodar localmente

Abra `index.html` direto no navegador ou sirva a pasta com um servidor estático, por exemplo:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Estrutura

- `index.html`
- `css/styles.css`
- `js/script.js`
- `js/log-lines.js`
